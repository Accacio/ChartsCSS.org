# Motion Effects

Motion effects enhance your charts when users interact with elements using mouse or touch events. The simplest example is CSS hover effects. You can add motion effects when the mouse hovers over the entire chart or an individual inner element. Use your imagination and CSS skills to create beautiful interaction effects.

Here are some basic examples leveraging CSS :hover pseudo-class.

# Data Hover Effect

A simple background color change when the user hovers over data items:

#motion-effect tr {
  transition-duration: 0.3s;
}
#motion-effect tr:hover {
  background-color: rgba(0, 0, 0, 0.05);
}

# Dataset Opacity Effect

Reverse opacity effect when hovering over datasets:

#motion-effect td {
  transition-duration: 0.3s;
  opacity: 0.5;
}
#motion-effect td:hover {
  opacity: 1;
}

# Scale Data

You can scale the data when hovering over the chart.

#motion-effect td .data {
  transition-duration: 0.6s;
  transform: scale(0);
}
#motion-effect td:hover .data {
  transform: scale(1);
}

# Grayscale Chart

My favorite effect is the grayscale effect. The chart becomes colorful only when the user hovers over chart.

#motion-effect {
  transition-duration: 1s;
  filter: grayscale(100%);
}
#motion-effect:hover {
  filter: none;
}
